# Architex

Architex is a full-stack web application for exploring architectural styles throughout history. Users can browse a curated collection of 16+ styles — from Gothic to Deconstructivism — view their key characteristics, time periods, and famous examples, and search by keyword. Registered users can save styles to their favorites and receive personalized recommendations based on time-period overlap with the styles they already like. The app also supports full CRUD operations, allowing authenticated users to add, edit, and delete architectural styles.

**Tech stack:** React · Flask · PostgreSQL · SQLAlchemy · JWT Authentication · Docker
